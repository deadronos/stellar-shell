# Architecture Index

This folder contains high-level architecture docs, technical specs, decision records and game design specs.

Conventions:
- High-level notes live in this file (`/docs/ARCHITECTURE.md`).
- Detailed artifacts live in `/docs/ARCHITECTURE/`.
- File prefixes:
  - Technical: `TEC###-*.md` (e.g., `TEC001-rendering-architecture.md`)
  - Decision record: `DEC###-*.md` (e.g., `DEC001-chunk-meshing.md`)
  - Game design: `GAME###-*.md` (e.g., `GAME001-drone-mechanics.md`)

See `/docs/ARCHITECTURE/TEC001-rendering-architecture.md` for an initial technical spec.
